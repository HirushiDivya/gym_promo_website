# Fitness Center - Premier Fitness Branding Website

A high-performance, fully responsive promotional website built for a local fitness brand. This project features a modern UI with dynamic theme switching, smooth scroll navigation, and integrated communication tools.

---

## 🛠️ Tech Stack

* **Frontend Library:** React.js (Functional Components & Hooks)
* **Styling:** Custom CSS3 (CSS Variables, Media Queries, Glassmorphism)
* **Icons:** Lucide-React
* **Alerts/Notifications:** SweetAlert2
* **Build Tool:** Vite (Next-Generation Frontend Tooling)

---

## 📥 Setup Instructions

Follow these steps to run the project on your local machine:

### 1. Clone the Repository
```bash
git clone [https://github.com/HirushiDivya/fitness-website.git](https://github.com/HirushiDivya/fitness-website.git)
```
cd fitness-website
npm install
npm start
